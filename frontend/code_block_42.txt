import React, { useEffect, useState } from 'react'; import { getAllUsers } from '../services/api'; import UserList from './UserList'; const Dashboard = ({ token }) => { const [users, setUsers] = useState([]); useEffect(() => { const fetchUsers = async () => { const res = await getAllUsers(token); setUsers(res.data.users); }; fetchUsers(); }, [token]); return ( <div> <h2>Admin Dashboard</h2> <UserList users={users} token={token} /> </div> ); }; export default Dashboard;
